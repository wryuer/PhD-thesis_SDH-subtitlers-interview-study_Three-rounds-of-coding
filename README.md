This repository contains three rounds of coding developed for Phase I of my PhD research on "From Production to Reception: Advancing Subtitling for the Deaf and Hard of Hearing in China".
The codes were generated from semi-structured interviews with nine SDH subtitlers working in different organisations. The data were analysed using MAXQDA.
Please cite the associated thesis or publication when using or referring to these materials.
